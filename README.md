# Drone Failure Prediction System

This project focuses on predicting drone failures using machine learning techniques.

## Description
The system analyzes drone flight data such as battery level, wind speed, payload, and GPS accuracy to detect potential failures.

## Features
- Data preprocessing and cleaning  
- Feature analysis and visualization  
- Machine learning model (ANN)  
- Performance evaluation (Accuracy, F1-score)  

## Technologies Used
- Python  
- Machine Learning  
- Data Analysis  
- Pandas, NumPy, Matplotlib
